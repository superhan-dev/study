# 풀이

문자열을 맞출때 stack과 같은 자료구조를 사용하면 닫히지 않았을 때 계속해서 push를 하고 마지막에 닫힌 괄호가 나오지 않거나 나왔지만 스택안에 남아있는 괄호가 있는 경우 실패를 하게 된다.

`(())()` 가 있다고 하자.

1. `(` 닫히지 않았다. -> push를 한다.
2. `(` 닫히지 않았다. -> push를 한다.
3. `)`를 사용하면 top에 있는 괄호를 닫을 수 있다. -> pop
4. `)`를 사용하면 top에 있는 괄호를 닫을 수 있다. -> pop
5. `(` 닫히지 않았다. -> push를 한다.
6. `)`를 사용하면 top에 있는 괄호를 닫을 수 있다. -> pop

남아있는 괄호가 없다. 'YES' 출력

# 코드

```javascript
const fs = require("fs");
function main() {
  let input = fs.readFileSync(0, "utf-8").trim().split("\n");

  const n = input[0];
  const open = "(";
  const close = ")";
  for (let idx = 1; idx < n; idx++) {
    let line = input[idx];
    const stack = [];
    let answer = "YES";
    let str = line.split("");
    for (let s of str) {
      if (s === open) {
        stack.push(s);
      } else if (s === close) {
        const top = stack[stack.length - 1];
        if (top === open) {
          stack.pop();
        } else {
          answer = "NO";
          break;
        }
      } else {
        answer = "NO";
        break;
      }
    }
    console.log(answer);
  }
}
```
